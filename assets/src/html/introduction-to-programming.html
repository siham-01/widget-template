Introduction to Programming

You've set the scene for your story. But, nothing happens in it yet.

Today, you'll program your scene to make it come to life. To program means to write step-by-step instructions for a computer.

These instructions need to be in a special language: a programming language.

You'll learn about one programming language today: CoBlocks.

Combining CoBlocks

CoBlocks can be combined into a sequence of CoBlocks.

You can read sequences from top to bottom, one block at a time.

A sequence of CoBlocks is also called a program. Let's look at one.

What do you think this program does?

Correct! Programs run from top to bottom, so the cat first moves 1 meter forward in 1 second, then changes to the lie down animation, and finally says "Meow!"
Not quite. Remember, CoBlocks run in order from top to bottom. What happens first? What happens second? What happens last?
